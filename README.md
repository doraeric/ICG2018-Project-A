# Interactive Computer Graphics
## Project A - Parking

### 操作說明

`↑`: 加速

`↓`: 後退

`→`: 右轉

`←`: 左轉

`Space`: 煞車

`R`: 重設車子位置到起點

`Z`: 開始倒車入庫

### 遊戲特色

具有自動回正方向盤與緩慢減速的功能，讓使用者更容易操作。

遊戲中有個小彩蛋，可以一起來找！

Hint: 和車子的速度有關喔OwO
